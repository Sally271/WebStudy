간단 linux 명령어

cd dir
change directory.
ex) cd .. -> 한 단계 상위 폴더로 이동
ex) cd desktop/WebStudy -> 바탕화면의 WebStudy 폴더로 이동

ls
list directory contents.
ex) ls -al -> -a: all, -l: list(파일 정보를 상세히 보여줌)
list가 너무 길어서 쭉쭉 내리고 (END)가 나오는 경우 q를 누르면 exit가능

echo "contents" > file
echo는 원래 텍스트나 문자열을 그대로 보여주는(출력하는) 명령어.
ex) echo "Geeks for Geeks" -> 터미널에 Geeks for Geeks 가 출력됨.
리눅스 커널에서 간단한 파일 생성 저장용으로 유용히 쓰일 수 있음.
ex) echo "hello world" > index.html -> index.html 파일을 만들고 거기에 hello world가 쓰임.

cat file
concatenate. 파일 이름을 인자로 받아서 그 내용을 쭉 이어주는 역할을 한다.
즉, 어떤 내용을 그냥 그대로 터미널 화면에 뿌려주는 역할이다.
ex) cat index.html -> index.html의 내용을 보여줌. (literally)

mv file1 file2
move. 말그대로 파일을 옮기는 역할을 하는 명령어인데
파일 이름을 바꾸고 싶을 때 쓸 수 있다. 위와 같이 쓰이면 file1이 file2로 복제되고 file1이 영구삭제됨.
ex) mv index.html Git.html -> index.html이 Git.html로 이름이 바뀌는 효과.

mkdir directory name
디렉토리(폴더)를 만든다.

code file
해당 파일을 VS Code로 연다.
